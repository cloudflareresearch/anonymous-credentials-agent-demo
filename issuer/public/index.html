Frequently Asked Questions

What is the main use case?
Enables to rate limit users trying to access a resource.
Does the user's identity gets protected?
Yes, server cannot determine which user is spending credits.
How does the two-step process work?
Simply request credits first, then spend them as needed. The process is streamlined for efficiency.
Can I revoke tokens after issuance?
Yes, ACT tokens provides token revocation capabilities. The credit value of revoked tokens is set to zero.

About this Website

This simple demo website that shows how a credit system can be used in a browser environment. The same functionality can be applied to a service fetching resources from an API endpoint.

Cloudflare Research team is currently investigating about anonymous credential systems.
